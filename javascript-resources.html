JavaScript is used in website development. It's for controlling behavior of your website. Like if you wanted to make a website that was a calculator, you'd need to use JavaScript. It's pretty advanced and difficult. I recommend learning HTML and CSS, then JavaScript if you're interested in website development. My background in Java has made JavaScript easier to learn, so learning Java beforehand will help, but isn't necessary.
